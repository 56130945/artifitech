# ArtifiTech Website

A modern, responsive website for ArtifiTech - showcasing web design and digital services.

## Features

- Responsive Design
- Modern UI/UX
- Service Showcase
- Portfolio/Projects Section
- Team Profiles
- Contact Form
- Testimonials
- Interactive Features:
  - AI-powered Chatbot Support
  - Live Product Demos
  - Interactive Product Tours
  - Real-time Support

## Technologies Used

- HTML5
- CSS3/SCSS
- JavaScript
- Bootstrap
- jQuery
- Font Awesome
- Dialogflow (Chatbot)
- Shepherd.js (Product Tours)

## Getting Started

1. Clone the repository
```bash
git clone https://github.com/yourusername/artifitech-website.git
```

2. Open index.html in your browser or set up a local server

## Project Structure

```
artifitech-website/
├── index.html          # Home page
├── about.html          # About page
├── service.html        # Services page
├── project.html        # Portfolio/Projects
├── team.html          # Team members
├── contact.html       # Contact page
├── css/               # Compiled CSS files
├── scss/              # SCSS source files
├── js/                # JavaScript files
│   ├── chatbot.js     # Chatbot integration
│   ├── product-tour.js # Shepherd.js implementation
├── img/               # Image assets
└── lib/               # Third-party libraries
```
